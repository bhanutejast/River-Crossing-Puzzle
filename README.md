# River-Crossing-Puzzle
A classic 🧩 AI river crossing puzzle where a farmer 🚣 must transport a wolf 🐺, goat 🐐, and cabbage 🥬 across a river. The boat carries only one at a time. Constraints: wolf eats goat, goat eats cabbage if left alone ⚠️. Goal: safely move all to the other side using valid moves ✅🚀
